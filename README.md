KGI RoleMind AI – Personal Agentic AI Chatbot Platform

KGI RoleMind AI is a smart chatbot system that allows users to create and use different AI assistants for different roles.
This chatbot is not a normal bot — it uses Agentic AI, meaning it can think, plan tasks, choose tools, and complete steps automatically based on the role selected.

Key Features
Create custom AI roles (Travel Planner, Coding Tutor, Placement Guide, Medical Info Assistant, etc.)
Switch between roles anytime
Uses latest AI models
Can fetch live internet information
Understands tasks and performs multi-step actions
Simple interface for chatting and role selection

Tools/Technologies Used
FastAPI (backend)
Streamlit (frontend)
LangChain & LangGraph (AI reasoning)
Groq + GPT models
Python

How it Works
User selects role
System loads the agent for that role
AI understands query
AI plans steps and uses tools
AI generates role-specific answer

Example Roles you can select
Travel Planner – creates trip plans
Placement Guide – mock interviews & resume help
Coding Tutor – explains code & fixes errors
Medical Info Assistant – general health information

Results
Test Role	Outcome
Travel Planner	Correct and updated trip plan
Coding Tutor	Accurate code help
Placement Guide	Good career and interview feedback
